I'm pausing Python. I've started the Google Data Analytics Certificate instead.

The reason is simple. I'm building a tool that reads a man's life and finds the patterns in it — what's rising, what's breaking, what he keeps avoiding. That work is data analysis. I've been building it without the skill it's made of. So I'm going to learn the skill.

Most people learn it on a sample spreadsheet and forget it by Friday. I'm going to learn it on the one dataset I can't lie to: my own life — the same days the tool already reads.

I'm at the start. I'll do it in the open, results or not.
